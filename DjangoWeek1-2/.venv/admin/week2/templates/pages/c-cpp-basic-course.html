Welcome to the C++ Basics Tutorial

Introduction

C++ is a powerful and widely used programming language that provides both low-level and high-level features. This tutorial will guide you through the fundamental concepts of C++ programming to help you get started.

Getting Started

Before you begin coding in C++, make sure you have a C++ compiler installed on your computer. You can download one such as GCC or use an integrated development environment (IDE) like Visual Studio or Code::Blocks.

Your First C++ Program

Create a new C++ source file (e.g., hello.cpp) and add the following code:


#include <iostream>

int main() {
    std::cout << "Hello, C++!" << std::endl;
    return 0;
}

Compile and run the program using your C++ compiler to see the output.

Variables and Data Types

C++ is a statically typed language, and you need to declare variables with their data types:


#include <iostream>

int main() {
    int number = 42;
    double pi = 3.14;
    char grade = 'A';

    return 0;
}

Control Flow

Use if, else, and else if statements for control flow:


#include <iostream>

int main() {
    int number = 42;

    if (number > 0) {
        std::cout << "Positive number" << std::endl;
    } else if (number == 0) {
        std::cout << "Zero" << std::endl;
    } else {
        std::cout << "Negative number" << std::endl;
    }

    return 0;
}

Loops

C++ supports for and while loops:


#include <iostream>

int main() {
    for (int i = 0; i < 5; i++) {
        std::cout << i << std::endl;
    }

    // or

    int counter = 0;
    while (counter < 5) {
        std::cout << counter << std::endl;
        counter++;
    }

    return 0;
}

Functions

Define functions using the void keyword for functions that do not return a value:


#include <iostream>

void greet(const std::string &name) {
    std::cout << "Hello, " << name << "!" << std::endl;
}

int main() {
    greet("John");
    return 0;
}

That's It!

Congratulations! You've completed a brief introduction to C++. There's a wealth of possibilities with C++, so continue exploring and building your programming skills.
