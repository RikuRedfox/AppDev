Welcome to the Python Basics Tutorial

Introduction

Python is a versatile and beginner-friendly programming language. This tutorial will guide you through the basics.

Getting Started

Before you start coding, make sure Python is installed on your computer. You can download it from the official Python website.

Your First Python Program

Create a new file with a .py extension (e.g., hello.py) and add the following code:


print("Hello, Python!")

Run the program by navigating to its directory in the terminal and typing python hello.py.

Variables and Data Types

In Python, you can declare variables without specifying their data types:


message = "Hello, Python!"
number = 42

Control Flow

Use if, else, and elif statements for control flow:


if number > 0:
  print("Positive number")
elif number == 0:
  print("Zero")
else:
  print("Negative number")

Loops

Python supports for and while loops:


for i in range(5):
  print(i)

# or

counter = 0
while counter < 5:
  print(counter)
  counter += 1

Functions

Define functions using the def keyword:


def greet(name):
  print("Hello, " + name + "!")

greet("John")

That's It!

Congratulations! You've completed a brief introduction to Python. There's much more to explore, so keep coding!
