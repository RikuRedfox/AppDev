Welcome to the JavaScript Basics Tutorial

Introduction

JavaScript is a dynamic and versatile programming language, well-suited for both beginners and experienced developers. This tutorial will walk you through the foundational concepts of JavaScript to help you grasp the basics.

Getting Started

If you're new to JavaScript, ensure that you have a text editor and a web browser installed. JavaScript is primarily used for web development and can be executed directly in the browser.

Your First JavaScript Program

Create an HTML file and add the following JavaScript code within the <script> tags:


console.log("Hello, JavaScript!");

Open the HTML file in your web browser, and you should see the output in the console.

Variables and Data Types

JavaScript is loosely typed, allowing you to declare variables without specifying their data types:


let message = "Hello, JavaScript!";
let number = 42;

Control Flow

Use if, else, and else if statements for control flow:


if (number > 0) {
  console.log("Positive number");
} else if (number === 0) {
  console.log("Zero");
} else {
  console.log("Negative number");
}

Loops

JavaScript supports for and while loops:


for (let i = 0; i < 5; i++) {
  console.log(i);
}

// or

let counter = 0;
while (counter < 5) {
  console.log(counter);
  counter++;
}

Functions

Define functions using the function keyword or using arrow functions:


function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("John");

That's It!

Congratulations! You've completed a brief introduction to JavaScript. There's a vast world of possibilities with JavaScript, so continue exploring and experimenting with your code!
