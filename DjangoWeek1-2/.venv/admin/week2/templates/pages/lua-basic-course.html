Welcome to the Lua Basics Tutorial

Introduction

Lua is a versatile scripting language designed for embedded systems and general-purpose programming. This tutorial will guide you through the fundamental concepts of Lua to help you start your journey.

Getting Started

To begin coding in Lua, you'll need to have a Lua interpreter installed on your system. You can download it from the official Lua website or use an online Lua interpreter for quick experimentation.

Your First Lua Program

Create a new Lua script file (e.g., hello.lua) and add the following code:


print("Hello, Lua!")

Run the script using the Lua interpreter to see the output.

Variables and Data Types

Like many scripting languages, Lua is dynamically typed, allowing you to declare variables without specifying their types:


message = "Hello, Lua!"
number = 42

Control Flow

Use if, else, and elseif statements for control flow:


number = 42

if number > 0 then
    print("Positive number")
elseif number == 0 then
    print("Zero")
else
    print("Negative number")
end

Loops

Lua supports for and while loops:


for i = 1, 5 do
    print(i)
end

-- or

counter = 0
while counter < 5 do
    print(counter)
    counter = counter + 1
end

Functions

Define functions using the function keyword:


function greet(name)
    print("Hello, " .. name .. "!")
end

greet("John")

That's It!

Congratulations! You've completed a brief introduction to Lua. Lua's simplicity and versatility make it an excellent choice for various applications. Keep exploring and experimenting with Lua to enhance your programming skills.
